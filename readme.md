# redSkeleton (Express)

```bash
npm i express fs-extra dotenv
npm i -D nodemon cross-env # 개발환경 
```

## 사용법
반드시 클로닝 이후에 npm install을 한번 실행해주세요.  
sample.config.env 를 config.en 로 바꿔주세요.


```sh
npm install # 처음 한번만
cp sample.config.env config.env
npm start
```
